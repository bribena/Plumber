Mivel létrehoz egy Layer-\/t, használat után a Pipe-\/ot fel kell szabadítani a \doxylink{pipe_8h_ad8e7c55fe9bad5b60e5c20b63f68cc5c}{Pipe\+\_\+\+Destroy()} függvénnyel. 
\end{DoxyPostcond}
